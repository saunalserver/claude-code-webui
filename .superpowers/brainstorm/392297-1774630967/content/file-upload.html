Feature: File Upload Sidebar

Upload files, get shareable URLs for AI see your context

Sidebar Panel
  • Drag-and drop images or files
  • Click to get shareable URL
  • URL auto-copies to clipboard
  • View uploaded files with thumbnails
  • Delete files when no longer needed

Storage Options

A

In-memory

Default. Fast. 48-hour TTL. Max 100 files. No persistence across restart.

B

disk volume

Persistent across restart. Survives restart. Unlimited size limit.
