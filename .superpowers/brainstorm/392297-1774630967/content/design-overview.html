Design: AI Terminal WebUI

Revamped multi-session terminal interface for AI CLIs

Project Goals

  • Portfolio showcase: Professional, polished code visible to recruiters
  • Genuinely usable: Others can self-host with their own AI CLI tools
  • AI-agnostic: Support Claude Code, Gemini CLI, Codex, and others via configuration

Core Features

  • Multi-session terminal tabs with xterm.js
  • Configurable AI CLI (claude, gemini, codex, custom)
  • File upload sidebar with shareable URLs
  • Session persistence (reconnect to active sessions)
  • Dark mode support

Technical Stack

  • Backend: Node.js, Express, WebSocket, node-pty
  • Frontend: xterm.js, Vanilla JS, CSS
  • Deployment: Docker with docker-compose

Key Changes from Current

  • Remove hardcoded personal info (Tailscale IP, paths, usernames)
  • Fix Docker user inconsistency (node vs saunalserver)
  • Add CLI configuration via environment variables
  • Modernize UI with clean SaaS design
  • Improve setup script for better UX
  • Add proper .gitignore for sensitive files
